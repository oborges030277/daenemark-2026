## Project Overview

A vacation planning website for a Denmark trip in 2026. Single-page German-language HTML page with embedded CSS/JS, generated by a Python build script. Currently in placeholder state — trip details (accommodation, route, beaches, restaurants etc.) are still to be filled in.

Schwester-Projekt: `../Urlaub 2026 - Kroatien` (gleiche Idee, dient als Referenz).

## Build

```bash
python build_final.py
```

Erzeugt `index.html` im Projektverzeichnis.

## Architecture

- **`build_final.py`** — Build-Script. Alle Daten (Unterkunft, Strände, Restaurants, Raststätten, Wetter, Ausflüge, Einkaufen) stehen als Python-Konstanten oben im Script. CSS und JS sind ebenfalls als Python-String-Konstanten enthalten. HTML wird per f-string zusammengebaut.
- **`index.html`** — Generated output. Nicht direkt editieren — stattdessen `build_final.py` anpassen und neu bauen.
- **`zugangsdaten.txt`** — GitHub-Zugangsdaten (per `.gitignore` ausgeschlossen).

## Struktur der Seite

Kategorien (gleiche wie beim Kroatien-Projekt):

1. **Hero** — Titel, Untertitel, Countdown
2. **Unterkunft** — Name, Lage, Ausstattung, Zimmer, Hausordnung
3. **Anreise** — Route, Maut/Fähren, Raststätten
4. **Wetter** — Durchschnittswerte
5. **Strände** — Karten mit Beschreibung, Route, Tags
6. **Ausflüge** — Sehenswürdigkeiten
7. **Restaurants** — Empfehlungen
8. **Einkaufen** — Supermärkte, Märkte
9. **Karte** — Übersichtskarte
